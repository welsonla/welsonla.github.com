blog with github(1)-前期准备

2013-05-10

##开始之前

  1. 安装Git
  2. 使用rbenv或者RVM安装ruby 1.9.3

安装完后确保你的ruby版本是1.9.3

ruby -v

##设置Octopress

git clone git://github.com/imathis/octopress.git octopress
cd octopress

接下来安装一些依赖

gem install bundler
rbenv rehash   
bundle install

安装Octopress默认主题

rake install
