### 安装Hexo-cli
```
yarn global add hexo-cli
```

### 安装Hexo-server
```
yarn add hexo-server
```

### 安装Hexo
```
yarn add hexo-deployer-git
```

### 安装RSS生成拓展
```
yarn add hexo-generator-feed
```


### 安装Hexo 主题
```
git submodule add git@github.com:welsonla/crisp-hexo-theme.git themes/crisp
```

### 更新submodule
```
git submodule init
git submodule update
```
